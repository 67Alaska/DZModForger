**License:** DayZ Modding Community

## 🎯 Overview

UltimateDZForge is a professional-grade 3D model editor specifically designed for DayZ object creation and modification. Built with DirectX 12 and WinUI 3, it provides Blender-like viewport controls, real-time rendering, and comprehensive model manipulation tools.

### Key Features

✅ **DirectX 12 Rendering**
- Real-time 3D viewport with 120+ FPS capability
- Professional shading modes (Wireframe, Solid, Material, Render)
- Grid, axes, and bounds visualization
- Orbital camera with Blender-style controls

✅ **Model Support**
- FBX (Autodesk FBX SDK 2020.3.4)
- OBJ (Wavefront)
- DAE (Collada/ZXML)
- Full geometry preservation (vertices, normals, UVs, tangents)

✅ **Professional Tools**
- Transform controls (position, rotation, scale)
- Material editor (metallic, roughness, ambient occlusion)
- Level of Detail (LOD) system
- Skeletal animation support
- Batch operations

✅ **Export Pipeline**
- DayZ config.cpp export
- FBX/OBJ/DAE export with format options
- Project save/load (JSON-based)
- Backup versioning system
